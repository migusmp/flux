# flux
Flux — TUI de monitorización y logs para DevOps

## Herramienta para monitorizar VPS
Flux es una herremienta TUI encargada de monitorizar procesos de un servidor VPS.
- Tendrá una funcion en la que se pueda activar que se ejecute en segundo plano y se puedan enviar los datos que vaya recogiendo o que le indiques a un backend o frontend determinado para ir monitorizandolo.
Se podrá configurar con un archivo .yaml o .config en el que le indiques que logs quieres que observe
